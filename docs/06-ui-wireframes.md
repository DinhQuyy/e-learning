# MÔ TẢ WIREFRAME GIAO DIỆN

## Hệ Thống Quản Lý Khoá Học Trực Tuyến (E-Learning Platform)

---

## 1. Tổng quan thiết kế

### 1.1. Design System

| Thuộc tính          | Giá trị                                          |
| ------------------- | ------------------------------------------------ |
| **Primary Color**   | Indigo (#4F46E5)                                 |
| **Secondary Color** | Slate (#64748B)                                  |
| **Success**         | Emerald (#10B981)                                |
| **Warning**         | Amber (#F59E0B)                                  |
| **Error**           | Red (#EF4444)                                    |
| **Background**      | White (#FFFFFF) / Slate-50 (#F8FAFC)             |
| **Font Family**     | Inter (heading + body)                           |
| **Border Radius**   | 8px (cards), 6px (inputs), 12px (large cards)    |
| **Shadow**          | shadow-sm, shadow-md (cards), shadow-lg (modals) |

### 1.2. Breakpoints (Responsive)

| Breakpoint | Width          | Layout                        |
| ---------- | -------------- | ----------------------------- |
| Mobile     | < 768px        | 1 column, hamburger menu      |
| Tablet     | 768px - 1023px | 2 columns, collapsed sidebar  |
| Desktop    | >= 1024px      | Full layout, expanded sidebar |

### 1.3. Layout chung

**Layout Components:**

- **PublicLayout:** Header + Main Content + Footer
- **AuthLayout:** Centered card, minimal header
- **DashboardLayout:** Header + Sidebar + Main Content (no footer)

## 8. Responsive Design Notes

### 8.1. Mobile Adaptations

| Component             | Desktop            | Mobile                         |
| --------------------- | ------------------ | ------------------------------ |
| Navigation            | Horizontal menu    | Hamburger + slide menu         |
| Dashboard Sidebar     | Fixed left sidebar | Hidden, slide from left        |
| Course Grid           | 3-4 columns        | 1 column (stacked)             |
| Course Player Sidebar | Fixed left         | Hidden, toggle button          |
| Data Tables           | Full columns       | Horizontal scroll or card view |
| Multi-step Form       | Horizontal stepper | Vertical stepper               |
| Filters               | Left sidebar       | Bottom sheet                   |
| Modals/Dialogs        | Center modal       | Full-screen sheet              |
| Stats Cards           | 4 in a row         | 2x2 grid                       |

### 8.2. Common UI Patterns

| Pattern                 | Mô tả                                           |
| ----------------------- | ----------------------------------------------- |
| **Skeleton Loading**    | Placeholder animation khi load data             |
| **Empty State**         | Illustration + message khi không có data        |
| **Error State**         | Error icon + message + retry button             |
| **Toast Notifications** | Bottom-right, auto dismiss sau 5s               |
| **Confirm Dialog**      | Modal xác nhận cho destructive actions          |
| **Infinite Scroll**     | Load more khi scroll (tuỳ chọn thay phân trang) |
| **Optimistic UI**       | Cập nhật UI ngay, rollback nếu API fail         |
